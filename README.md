# 猫猫熄屏

英文名：SleepyCat Screen Off

  <img src="assets/icon-cropped-preview.png" width="128" alt="猫猫熄屏图标">

一个本地运行的 macOS 小工具。双击 `猫猫熄屏.app` 后，所有已连接屏幕会进入真实显示器睡眠；电脑本身会被保持唤醒 8 小时，继续下载、计算、同步或运行本地任务。

## 下载使用

1. 在 GitHub Releases 下载 `SleepyCat-Screen-Off-macOS.zip`。
2. 解压后得到 `猫猫熄屏.app`。
3. 把 app 放到“应用程序”或桌面。
4. 双击运行。

如果 macOS 提示无法打开未验证开发者的 app，第一次可以右键点击 `猫猫熄屏.app`，选择“打开”。

如果把它固定到 Dock，单击一次即可。它是固定 8 小时版本，不是永久模式。

## 它做什么

- 让所有内置和外接显示器熄屏。
- 固定保持 Mac 本身继续运行 8 小时。
- 只有在熄屏请求成功后才启动保活进程。
- 鼠标或键盘激活后会自动清理并退出。
- 可以熄掉所有已连接的扩展屏，同时保持电脑正常运行，适合下载、长时间本地任务和 vibe coding。
- 不联网，不上传数据，不收集信息。
- 使用标准 AppleScript app 外壳调用 macOS 原生 `pmset displaysleepnow`，不使用黑屏遮罩兜底。

## 重要注意

- 建议插着电源使用。
- 不要合盖；Mac 笔记本合盖通常会触发系统睡眠。
- 这不是 macOS 的“睡眠”，也不是锁屏、屏保或视频播放器的熄屏按钮。
- 它只让显示器睡眠，同时让电脑在 8 小时内继续完成下载、计算、同步或科研任务。
- 不同 macOS 版本、外接显示器、扩展坞和电源设置可能有差异。
- 这是固定 8 小时版；8 小时后保持唤醒会自动结束。
- 如果有视频播放、浏览器视频唤醒锁或系统拒绝熄屏，程序会退出，不会留下保活残留。

## 从源码打包

维护者在项目根目录运行：

```bash
./scripts/build_release.sh
```

生成结果：

- `build/猫猫熄屏.app`
- `dist/SleepyCat-Screen-Off-macOS.zip`
